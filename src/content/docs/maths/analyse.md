# Les Développements Limités usuels

Voici la base à connaître par cœur pour le concours :
* $e^x = 1 + x + \frac{x^2}{2!} + o(x^2)$
* $\cos(x) = 1 - \frac{x^2}{2!} + o(x^2)$

**Astuce :** Toujours vérifier l'ordre demandé avant de commencer !
